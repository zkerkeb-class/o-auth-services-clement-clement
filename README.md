## Installation and configuration

Run `npm install`

Create `.env` file

.env configuration :

 <pre>
PORT=3001

DB_HOST=
DB_USER=
DB_PASSWORD=
DB_DATABASE=
DB_PORT=3306

JWT_SECRET=
 </pre>

## Run project

Run `npm run dev`
